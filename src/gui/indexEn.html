Top Row Inputs

Buttons
    New - Opens a new meal editor
    Open - Opens a recipe file
    Save - Saves to current file working in
    Save As - Saves the current recipe to specified file name
    Close - Closes current file
Text Box
    Name - Name of current recipe
    Serves - Amount of people the feeds

Utensils

Inputs
    Name - Name of utensil to be added
    Details - Details of recipe to be added
Buttons
    Add - Adds Utensil based on name and detail given
    Delete - Deletes selected utensil from recipe

Ingredients

Inputs
    Name - Name of ingredient to be added
    Details - Detail of ingredient to be added
    Amount - Amount of Ingredient in selected Unit
    Units - Amount of Selected units of ingredient
Buttons
    Add - Adds Ingredient based on name, detail, amount and unit given
    Delete - Deletes selected ingredient from recipe

Steps

Inputs
    Action - What to do to selected items
    On - Ingredient to do Action to
    Utensil - Utensil to use for step
    Details - Details about current step
    Time - Time to complete step
Buttons
    Add - Adds step based on action, ingredient, utensil, detail and time given
    Delete - Deletes selected step from recipe
Drink Pairing
    Drink - drink paired with recipe
