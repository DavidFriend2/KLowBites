Guía de Usuario de KILowBites
Archivo
    Salir - Cierra la aplicación actual
Editar
    Comida - Abre la ventana del editor de comidas
    Receta - Abre la ventana del editor de recetas
Herramientas
    Calculadora de Calorías - Abre la ventana del convertidor de calorías
    Convertidor de Unidades - Abre la ventana del convertidor de unidades
Búsqueda
    Buscador de Recetas - Abre el buscador de recetas
    Buscador de Comidas - Abre el buscador de comidas
Preferencias
    Idiomas - Lista de idiomas para seleccionar
Ayuda
    Guía de Usuario - Abre la página actual

Entradas de la Fila Superior

Botones
    Nuevo - Abre un nuevo editor de comidas
    Abrir - Abre un archivo de receta
    Guardar - Guarda en el archivo actual
    Guardar Como - Guarda la receta actual con un nombre de archivo especificado
    Cerrar - Cierra el archivo actual
Caja de Texto
    Nombre - Nombre de la receta actual
    Sirve - Cantidad de personas que sirve

Utensilios

Entradas
    Nombre - Nombre del utensilio a añadir
    Detalles - Detalles de la receta a añadir
Botones
    Agregar - Añade utensilio según el nombre y los detalles dados
    Eliminar - Elimina el utensilio seleccionado de la receta

Ingredientes

Entradas
    Nombre - Nombre del ingrediente a añadir
    Detalles - Detalles del ingrediente a añadir
    Cantidad - Cantidad del ingrediente en la unidad seleccionada
    Unidades - Unidades seleccionadas del ingrediente
Botones
    Agregar - Añade ingrediente según el nombre, detalle, cantidad y unidad dados
    Eliminar - Elimina el ingrediente seleccionado de la receta

Pasos

Entradas
    Acción - Qué hacer con los elementos seleccionados
    Sobre - Ingrediente al que hacer la acción
    Utensilio - Utensilio a utilizar para el paso
    Detalles - Detalles sobre el paso actual
    Tiempo - Tiempo para completar el paso
Botones
    Agregar - Añade paso basado en acción, ingrediente, utensilio, detalle y tiempo dados
    Eliminar - Elimina el paso seleccionado de la receta

Entradas de la Fila Superior

Botones
    Nuevo - Abre un nuevo editor de comidas
    Abrir - Abre un archivo de comida
    Guardar - Guarda en el archivo actual
    Guardar Como - Guarda la comida actual con un nombre de archivo especificado
    Cerrar - Cierra la ventana actual
Seleccionables
    Nombre - Nombre de la comida actual
Botones
    Agregar Receta - Añade una receta a la comida actual
    Crear Receta - Abre el editor de recetas para crear una nueva receta
    Eliminar - Elimina la receta seleccionada de la comida
Botones
    Calcular - Convierte los valores seleccionados a las unidades seleccionadas
    Restablecer - Restablece todos los cuadros de selección y valores
Seleccionable
    Unidades de Origen - Unidades a convertir
    Unidades de Destino - Unidades a las que se convertirá
    Ingredientes - Ingrediente a convertir
    Cantidad de Origen - Cantidad del ingrediente a convertir
Resultado
    Cantidad de Destino - Unidad convertida
Botones
    Calcular - Calcula las calorías de los valores seleccionados
    Restablecer - Restablece todos los cuadros de selección y valores
Seleccionable
    Ingrediente - Ingrediente para calcular
    Cantidad - Cantidad del ingrediente a calcular
Resultado
    Calorías - Cantidad de calorías según los valores seleccionados
Botones
    Elegir - seleccionar el directorio a buscar
    Agregar - Añadir ingredientes a buscar
    Agregar Ingredientes - Escribe el ingrediente para buscar
Selecciones
    Eliminar Ingrediente - Eliminar ingrediente de la lista de búsqueda
    Buscar Recetas - Buscar recetas con los ingredientes seleccionados
    Lista - Muestra los ingredientes en la receta seleccionada
    Cuadro desplegable - Seleccionar
Botones
    Elegir - seleccionar el directorio a buscar

Visualización
    Utensilios - Muestra los utensilios necesarios en la receta actual
    Pasos - Muestra la lista de pasos en orden
